# 6단계 — 1번 명언삭제

- 관련 강의: 11강 ~ 12강
- 상태: 진행중
- 시작일: 2026-07-16

## 요구사항 요약

`삭제?id=1` 형식으로 명령을 입력하면 해당 번호의 명언을 삭제한다.

```
== 명언 앱 ==
명령) 등록
명언 : 현재를 사랑하라.
작가 : 작자미상
1번 명언이 등록되었습니다.
명령) 등록
명언 : 과거에 집착하지 마라.
작가 : 작자미상
2번 명언이 등록되었습니다.
명령) 목록
번호 / 작가 / 명언
----------------------
2 / 작자미상 / 과거에 집착하지 마라.
1 / 작자미상 / 현재를 사랑하라.
명령) 삭제?id=1
1번 명언이 삭제되었습니다.
명령) 종료
```
