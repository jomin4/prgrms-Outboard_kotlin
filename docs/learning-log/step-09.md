# 9단계 — 파일을 통한 영속성

- 관련 강의: 27강 ~ 33강
- 상태: 완료
- 시작일: 2026-07-16
- 완료일: 2026-07-16

## 요구사항 요약

각 명언을 개별 파일로 `db/wiseSaying` 폴더에 저장한다.

- 파일명: `{명언번호}.json`
- 등록/수정 시마다 해당 파일 갱신
- 가장 마지막 명언 번호는 `db/wiseSaying/lastId.txt`에 저장
- 프로그램을 다시 시작해도 기존 데이터가 유지되어야 함

```json
{
  "id": 1,
  "content": "명언 1",
  "author": "작가 1"
}
```
